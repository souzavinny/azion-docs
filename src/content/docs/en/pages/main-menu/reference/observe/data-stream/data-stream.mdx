**Data Stream** feeds your stream processing, SIEM, and big data platforms with the event logs from your applications on Azion in real time, so you can understand user behavior, analyze performance, identify security threats, and make informed decisions.

Data Stream is an [Observe](/en/documentation/observe/overview/) product responsible for collecting, grouping, and transmitting raw logs to the destination you choose. You select a *data source* (the Azion product that generates the event logs), a *template* with the variables you want to collect, the *domains* whose events you want to capture, and an *endpoint* (the stream processing platform or analysis tool that receives the data). To keep an enhanced performance, it uses ASCII encoding to avoid parser issues and problems in data interpretation.

By default, logs are sent in batches when the block reaches 2,000 records or every 60 seconds, whichever occurs first, and the data reaches your infrastructure in up to 3 minutes. After configuring your stream, you can check your logs successfully being sent through [Real-Time Events](/en/documentation/observe/real-time-events/).

You can manage your streams through [Azion Console](https://console.azion.com/data-stream) or the [Azion API](https://api.azion.com/#04257075-3691-4b5f-b22b-22a5b6653f3f).

## Limits and plans

:::tip
**Increase limits** <br></br>
You can request to increase the limits based on your plan. Contact the [technical support team](/en/documentation/services/support/) to request it.
:::

These are the **default limits**:

| Scope | Limit |
| ------- | ----- |
| Data Stream | 60 seconds or packages of 2,000 requests, whichever happens first.<br /><br />For the **AWS Kinesis Data Firehose** endpoint: 60 seconds or packages of 500 requests |
| Sampling | 1 active stream |
| Time for data transfer to client's infrastructure | Up to 3 minutes |

Data Stream is a billed product. Make sure you have it activated in your account: [contact the sales team](https://www.azion.com/en/contact-sales/) for more details, and check the [pricing page](/en/documentation/platform/pricing/) for more information.
