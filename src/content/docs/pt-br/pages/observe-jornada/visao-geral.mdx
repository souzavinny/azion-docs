Os produtos de Observe da Azion auxiliam você no monitoramento e acesso a informações relacionadas aos seus dados. Você pode analisá-los na Azion Web Platform ou conectar seus dados a plataformas de terceiros, melhorando sua compreensão do seu conteúdo, como seus usuários estão acessando elas, obtendo insights sobre desempenho e segurança e tomando decisões mais informadas.

## Como funciona

### Real-Time Metrics

Monitore aplicações em tempo real visualizando gráficos e filtrando suas consultas.

Você também pode usar a [API GraphQL](/pt-br/documentacao/devtools/graphql/reference/overview/) para consultar seus dados via API e personalizar suas consultas.

![Visão geral do fluxo de uma consulta no Real-Time Metrics](/assets/docs/images/uploads/real-time-metrics-overview.png)

<LinkButton link="/pt-br/documentacao/observe/data-stream/guides/monitorar-aplicacoes/" label="ir para monitorar aplicações" severity="secondary" target="_blank" />

### Real-Time Events

Analise logs em tempo real por meio de listas detalhadas de variáveis para entender e diagnosticar comportamentos.

Você também pode usar a [API GraphQL](/pt-br/documentacao/devtools/graphql/reference/overview/) para consultar seus dados via API, personalizar suas consultas e criar consultas top X.

![Visão geral do fluxo de uma consulta no Real-Time Events](/assets/docs/images/uploads/real-time-events-overview.png)

<LinkButton link="/pt-br/documentacao/observe/data-stream/guides/analisar-logs/" label="ir para analisar logs" severity="secondary" target="_blank" />

### Data Stream

Transmita seus dados em tempo real para plataformas de terceiros para ajudar a entender e analisar o desempenho e comportamento de sua application e identificar ameaças de segurança.

![Visão geral do fluxo do Data Stream](/assets/docs/images/uploads/data-stream-overview.png)

<LinkButton link="/pt-br/documentacao/observe/data-stream/guides/transmitir-dados/" label="ir para transmitir dados" severity="secondary" target="_blank" />

### Edge Pulse

Colete dados de navegação adicionando tags nas páginas do seu navegador para entender como seu conteúdo está sendo acessado e melhorar a experiência do usuário.

<LinkButton link="/pt-br/documentacao/observe/data-stream/guides/analisar-dados-navegacao/" label="ir coletar dados de navegação" severity="secondary" target="_blank" /><br />
